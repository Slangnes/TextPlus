TextPlus Demo Games

Learn by Playing — Progressive Tutorial for TextPlus Core

How to Use These Demos

TextPlus is a framework for creating interactive fiction games. These three demo games are designed to teach you how to use the TextPlus Core API by example.

Play in order: Start with Hello World, progress to The Detective's Case, then try The Memory Keeper. Each game demonstrates more advanced features than the last.

As you play, look at the game code in your editor (game.ts and index.html) to understand how features like save/load, event listeners, and theming work in practice.

Hello World

Level 1 - Beginner
A minimal interactive fiction game that teaches the fundamentals.
  • Situation transitions
  • Quality tracking
  • Conditional text
  • Multiple endings
  • DOM rendering

Perfect starting point. Demonstrates core concept: situations as game states, qualities as player attributes, links as choices.

Core API Features Demonstrated

Level 1: Hello World

Engine: createGame, goToSituation, followLink
State: getQuality, setQuality, mutateQuality
Info: getCurrentSituation, getSituation

Level 2: Detective Case

Events: onQualityChange, onSituationChange
History: getSituationHistory, hasSituationBeenVisited
Persistence: getSaveState, loadState

Level 3: Memory Keeper

Theming: applyTheme, theme CSS variables
State: Quality constraints (min/max)
Lifecycle: onEnter/onExit callbacks, reset

Recommended Learning Path

  1. Play Hello World — Understand the basic model: situations, qualities, links. Open hello-world/game.ts and read the code while playing.
  2. Read Detective Case code — Look at detective-case/game.ts. Notice the event listeners in index.html. See how `onQualityChange` tracks investigation progress in real-time.
  3. Play Detective Case — Experience save/load. Notice how your investigation score and evidence change based on your choices.
  4. Read Memory Keeper code — Study how quality constraints work. See how the theme changes based on sanity level.
  5. Play Memory Keeper — Experience theme switching and complex state management. Experiment with save slots.
  6. Build your own game — Use one of these as a template. Copy the folder, modify game.ts and index.html, and create your own interactive fiction!
